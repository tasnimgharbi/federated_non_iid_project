# Federated Learning on Non-IID Data

## Big Data Management Project

**Student:** Tasnim Gharbi

**Professor:** Francesco La Rosa

**University:** University of Messina

### Description

This project investigates the impact of non-IID data distributions on Federated Learning.

Implemented algorithms:
- FedAvg
- FedProx
- FedNova-inspired normalized aggregation

Datasets:
- Fashion-MNIST
- Human Activity Recognition (HAR)

Partitioning strategies:
- IID
- Label-skew
- Quantity-skew

Evaluation metrics:
- Global Accuracy
- Training Loss
- Client Accuracy Standard Deviation
- Client Accuracy Variance
- Communication Cost
- Simulated Network Latency

Framework:
- PyTorch
